有一次不小心在操作 PowerShell 7 的時候，無意間點到 Alt + Shift + D
發現原來 PowerShell 7 有快捷鍵這東西，這篇文章紀錄一下這個動作的開關以及如何設定

# 常用快捷鍵

  • Alt + Shift + + 垂直分割
  • Alt + Shift + - 水平分割
  • Alt + Shift + D 自動分割 (以長邊)
  • Alt + ↑ ↓ ← → 切換焦點
  • Alt + Shift + ↑ ↓ ← → 調整窗格大小
  • Ctrl + Shift + w 關閉窗格
