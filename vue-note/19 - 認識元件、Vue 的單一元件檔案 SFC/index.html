# иӘҚиӯҳе…ғд»¶

еңЁ Vue е№«жҲ‘еҖ‘и§Јжұәзҡ„дәӢжғ…йҖҷзҜҮж–Үз« дёӯпјҢе·Із¶“з”ЁдёҖеҖӢз°Ўе–®зҡ„ж–№ејҸд»Ӣзҙ№е…ғд»¶зҡ„жҰӮеҝөгҖӮ

з•¶жҷӮеҸӘжҳҜжҸҗеҲ°жЁЎжқҝзөҗж§Ӣи·ҹиіҮж–ҷжҠҪйӣўпјҢдёҰдё”и®“зөҗж§ӢйҮҚиӨҮдҪҝз”ЁпјҢеё¶дёҠдёҚеҗҢзҡ„иіҮж–ҷгҖӮ

жңүжІ’жңүиҰәеҫ—йҖҷжқұиҘҝи·ҹд»ҖйәјеҫҲеғҸпјҹ

е°ҚвҖҰ е°ұжҳҜдёҠдёҖзҜҮж–Үз« зҡ„ v-for еңЁеҒҡзҡ„дәӢжғ…гҖӮ

еҜҰйҡӣдёҠйҖҷеҸӘжҳҜдёҖеҖӢжңҖж·әзҡ„жҰӮеҝөдҫҶи®“ж–°жүӢеҺ»гҖҢж„ҹеҸ—гҖҚд»ҖйәјеҸ«еҒҡе…ғд»¶пјҢдҪҶ v-for еңЁеҒҡзҡ„дәӢжғ…пјҢзҙ”зІ№еҸӘжҳҜи®“з•«йқўйҮҚиӨҮиў«жёІжҹ“пјҢеҡҙж јиӘӘиө·дҫҶдёҚиғҪзЁұд№ӢзӮәе…ғд»¶гҖӮ

Vue зҡ„е…ғд»¶пјҢйҷӨдәҶиғҪи®“жЁЎжқҝгҖҢйҮҚиӨҮдҪҝз”ЁгҖҚд№ӢеӨ–пјҢжӣҙйҮҚиҰҒзҡ„йӮ„жҳҜиғҪеңЁж•ҙеҖӢ Vue жҮүз”ЁзЁӢејҸдёӯзҡ„д»»дҪ•дҪҚзҪ®дҪҝз”ЁгҖӮ

иҖҢдё”йҷӨдәҶгҖҢз•«йқўгҖҚд№ӢеӨ–пјҢйӮ„жңғеҸҜд»ҘеҢ…еҗ«дёҖдәӣиіҮж–ҷеҠ е·ҘиҲҮйӮҸијҜж•ҙеҗҲгҖӮ

жҲ‘еҖ‘еүӣеӯёзҝ’з¶Ій Ғзҡ„й–Ӣзҷјж–№ејҸпјҢе°ұжҳҜдҪ жҲ‘йғҪзҶҹзҹҘзҡ„дёүеҖӢжӘ”жЎҲпјҡ index.html гҖҒ app.js гҖҒ style.css зҡ„еҒҡжі•пјҢеҹәжң¬дёҠе·Із¶“еҒҡеҲ°иЎЁзҸҫеұӨзҙҡзҡ„й—ңжіЁй»һеҲҶйӣўпјҢдёҚйҒҺеҸӘиҰҒе–®дёҖй Ғйқўзҡ„иіҮж–ҷйҮҸйҒҺж–јйҫҗеӨ§пјҢдёҠйқўиӘӘзҡ„йҖҷдёүеҖӢжӘ”жЎҲе°ұжҳҜз„ЎжӯўзӣЎзҡ„и®ҠеӨ§пјҢзңӢеҲ°е№ҫзҷҫиЎҢеҺҹе§Ӣзўјз®—жҳҜе®¶еёёдҫҝйЈҜпјҢжӣҙиӘҮејөзҡ„йӮ„еҸҜиғҪжңғеҲ°е№ҫеҚғиЎҢгҖӮ

йӣ–з„¶йҖҷжЁЈеҒҡдёҚжҳҜдёҚеҸҜиЎҢпјҢдёҚйҒҺжҷӮй–“дёҚз”ЁеӨӘд№…пјҢеҜ«е®ҢзЁӢејҸдёҖеҖӢжңҲеҫҢпјҢйҒҮеҲ°йңҖжұӮдҝ®ж”№пјҢиҰҒеҫһгҖҢзЁӢејҸеұұгҖҚдёӯжүҫеҲ°иҰҒж”№зҡ„ең°ж–№зө•е°ҚдёҚжңғжҳҜдёҖд»¶е®№жҳ“зҡ„дәӢжғ…гҖӮ

ж–јжҳҜжҲ‘еҖ‘еҸҜд»ҘжҠҠз¶Ій Ғз•«йқўз•¶жҲҗжӢјең–дёҖжЁЈеҲҮеүІпјҢHeader з•¶дҪңдёҖеЎҠпјҢдё»иҰҒе…§е®№з•¶жҲҗеҸҰдёҖеЎҠпјҢFooter еҶҚеҲҮдёҖеЎҠпјҢжӢҶжҲҗдёҚеҗҢзҡ„еҚҖеЎҠзңӢеҫ…гҖӮ

з”ҡиҮійӮ„еҸҜд»Ҙдҫқж“ҡеҗ„еҚҖеЎҠе…§е®№зҡ„зҚЁзү№жҖ§пјҢеҶҚи®“жӢјең–зҡ„еӨ§е°Ҹжӣҙе°ҸдёҖдәӣгҖӮ

жңҖеҫҢжҠҠйҖҷдәӣе…§е®№ж•ҙзҗҶжҲҗдёҖеҖӢдёҖеҖӢзҡ„е…ғд»¶жӘ”жЎҲпјҢжҲ‘еҖ‘зЁұдҪң е–®дёҖе…ғд»¶жӘ” SCF (sigle component file)гҖӮ

# SCF е–®дёҖе…ғд»¶жӘ”жЎҲ

Vue жҳҜдёҖеҖӢжјёйҖІејҸжЎҶжһ¶пјҢеүҚе№ҫзҜҮз”ЁеҲ°зҡ„зЁӢејҸеҜ«жі•д№ҹйғҪеҸҜд»Ҙз”ЁдҫҶе®ҡзҫ©е…ғд»¶пјҢй җиЁҲдёӢдёҖзҜҮж–Үз« жңғеҜ«еҲ°гҖӮ

дёҚйҒҺе…ғд»¶жңҖж–№дҫҝпјҢйӮ„жҳҜзӣҙжҺҘдҪҝз”Ё Vue зҡ„и…іжүӢжһ¶ (vite) жҷӮпјҢеҸҜд»ҘзӣҙжҺҘдҪҝз”Ё .vue еүҜжӘ”еҗҚзҡ„жӘ”жЎҲпјҢжҜҸдёҖеҖӢйҖҷзЁ®жӘ”жЎҲпјҢжҲ‘еҖ‘е°ұеҸҜд»ҘзЁұд»–еҸ«еҒҡе–®дёҖе…ғд»¶жӘ”жЎҲгҖӮ

  1. script setup жЁҷзұӨпјҡ

д»–е°ұжҳҜжҲ‘еҖ‘еҜ« setup () зҡ„ең°ж–№пјҢйӮЈеҖӢ setup жҳҜ Vue жҸҗдҫӣзҡ„иӘһжі•зі–пјҢе°ӨйӣЁжәӘеӨ§еӨ§жӣҫиӘӘйҖҷеҫҲй»‘йӯ”жі•пјҢд»–еҸҜд»Ҙи®“жҲ‘еҖ‘зӣҙжҺҘзңҒз•Ҙ setup () еҮҪж•ёпјҢиҖҢдё”и®Ҡж•ёдёҚз”ЁеҶҚ return еҮәдҫҶпјҢжЁЎжқҝеҚіеҸҜдҪҝз”ЁгҖӮ

д№ҹе°ұжҳҜеҰӮжһңеҸӘжңүеҜ« <script></script> пјҢе°ұйӮ„жҳҜеҝ…й ҲеҜ«йҖҷжЁЈзҡ„е…§е®№пјҡ

жңүдәҶ script setup иӘһжі•д№ӢеҫҢпјҢе°ұеҸҜд»ҘзңҒз•ҘжҲҗйҖҷжЁЈпјҡ

жңүзңӢеҲ°йӮЈеҖӢ import е—ҺпјҢжІ’йҢҜпјҒ Vue зҡ„ SCF жӘ”жЎҲйҒ©з”Ёж–ј ES Module иӘһжі•гҖӮ

  1. template жЁҷзұӨ

жЁЎжқҝпјҢд»–е°ұжҳҜжҲ‘еҖ‘еҜ« HTML зҡ„ең°ж–№пјҢи·ҹд№ӢеүҚзҡ„ж–Үз« зҜ„дҫӢдёӯпјҢеҜ«еңЁ #app иЈЎйқўзҡ„йӮЈдәӣжқұиҘҝдёҖжЁЈгҖӮ

з•¶з„¶пјҢеҰӮжһңдҪҝз”Ё v-if жҲ–жҳҜ v-for жҷӮжңүйңҖиҰҒпјҢйҖҷеҖӢжЁҷзұӨд№ҹйӮ„жҳҜеҸҜд»ҘеҶҚеҜ«дёҖеҖӢ template жЁҷзұӨпјҢе®Ңе…ЁжҳҜжІ’жңүе•ҸйЎҢзҡ„гҖӮ

  1. style scoped жЁҷзұӨ

е°ұжҳҜеҜ« CSS зҡ„ең°ж–№гҖӮ
