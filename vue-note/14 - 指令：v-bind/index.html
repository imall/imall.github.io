# Class 的綁定

一般 HTML 屬性的資料往往是「一個」值，但 class 的資料可能會出現好幾個值，所以 Vue 提供了幾種不同的寫法：

  1. 直接寫一個物件，並且把 class 名稱當作 key，值為布林值，當值為 true 時，該 class 會被加入，反之則不會。

這個寫法的好處是，我們只要去控制 isRed 變數值，就可以動態更改模板的 class 名稱。
