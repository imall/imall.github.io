# 常用快捷鍵
- `Alt` + `Shift` + `+` 垂直分割
- `Alt` + `Shift` + `-` 水平分割
- `Alt` + `Shift` + `D` 自動分割(以長邊)
- `Alt` + `↑` `↓` `←` `→` 切換焦點
- `Alt` + `Shift` + `↑` `↓` `←` `→` 調整窗格大小
- `Ctrl` + `Shift` + `w` 關閉窗格


# 快捷鍵設定
